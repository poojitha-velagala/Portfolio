Project

Clinical Data Analysis management system is web application which is used in hospitals to store patient’s details and extract their details easily. In this application admin can add the doctors in their respective hospitals and he/she can view the patient reports which are present in the data base . And also in this application doctor directly collects all the information of the patient regarding the patient’s name, id, age, gender , weight, blood group and the description of the disease. Doctors can analyse the disease of the patient easily within short time with this information . Doctor can be able to manage patient’s records and can do operations like Update , View , Delete. The different number of hospitals can be connected to this website and would be able to view the patients records from different hospitals which are stored in the database. When a doctor finds new symptoms he can add the related information to the database which he can analyse the same disease in the future without any waste of time.
